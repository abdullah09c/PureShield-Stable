# 🛡️ PureShield

**Guard Your Mind. Purify Your Screen.**

A minimal, lightweight Android app that blocks Reels & Shorts on social media and helps filter adult content via Private DNS — built with an Islamic theme.

---

## Features

- **Reels & Shorts Blocker** — YouTube Shorts, Facebook Reels, FB Lite, Instagram Reels, TikTok
- **Safe Internet (Private DNS)** — CleanBrowsing Family/Adult, Cloudflare Family, Google Safe
- **Uninstall Protection** — PIN-protected Device Admin to prevent impulsive removal
- **Start on Boot** — Automatically activates when phone turns on
- **Persistent Notification** — Always visible, low priority
- **Custom Block Message** — Edit the Ayah or reminder shown on the block screen
- **100% Private** — No internet permission, no data collected, no tracking

---

## Setup in Android Studio

1. **Open project**: `File → Open → select PureShield folder`
2. **Let Gradle sync** (first time downloads ~500MB of SDK tools)
3. **Connect your phone**: Enable USB Debugging (Settings → Developer Options)
4. **Run**: Click ▶ (Run button) — installs directly to your phone

---

## First Launch on Phone

1. Open PureShield
2. Allow **battery optimization exemption** (tap Disable Optimization)
3. Allow **notification permission** (Android 13+)
4. Tap the **Reels Blocker toggle** → it opens Accessibility Settings
5. Find **PureShield** in the list → enable it
6. ⚠️ Android 13+: If you see "Restricted Setting" tap ⋮ → Allow Restricted Settings first
7. Return to app — toggle turns green ✅

## Tech Stack

- **Language**: Kotlin
- **UI**: XML layouts (Material Components)
- **Min SDK**: 21 (Android 5.0) — supports ~99% of devices
- **Dependencies**: AndroidX only — no third-party libraries
- **APK size**: ~2–3 MB
